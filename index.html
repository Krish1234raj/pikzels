Installation Guide

Resource Pack Installation:

  1. Download the resource pack
  2. Open Minecraft and go to Options
  3. Select Resource Packs
  4. Click "Open Pack Folder"
  5. Move the downloaded file to this folder
  6. Activate the pack in-game

Modpack Installation:

  1. Install Minecraft Forge 1.20.1
  2. Download our modpack
  3. Extract to your .minecraft folder
  4. Launch using Forge profile
  5. Connect to play.pikzels.com
  6. Enjoy enhanced gameplay!

Frequently Asked Questions

Simply add our server IP "play.pikzels.com" to your Minecraft server list and join! We support Minecraft versions 1.19.x to 1.20.x. Make sure you have a stable internet connection for the best experience.

We support Minecraft Java Edition versions 1.19.x to 1.20.x. We recommend using the latest version (1.20.1) for the best performance and access to all features.

You can report players by using the "/report [playername] [reason]" command in-game, or by creating a ticket in our Discord server with evidence (screenshots/videos). Our moderation team reviews all reports within 24 hours.

Refunds are available within 7 days of purchase if you haven't used any of the purchased items/ranks. Contact our support team via email at support@pikzels.com with your purchase details for refund requests.

To appeal a ban, create a ticket in our Discord server or send an email to appeals@pikzels.com. Include your Minecraft username, ban reason, and a detailed explanation. Appeals are reviewed within 48 hours by our administration team.
