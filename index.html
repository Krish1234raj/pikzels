Installation Guide

Resource Pack Installation

  1. 1. Download the resource pack
  2. 2. Open Minecraft and go to Options
  3. 3. Select Resource Packs
  4. 4. Click "Open Pack Folder"
  5. 5. Move the downloaded file to this folder
  6. 6. Select the pack in-game and click "Done"

Modpack Installation

  1. 1. Install Minecraft Forge 1.20.1
  2. 2. Download and run OG MODPACK installer
  3. 3. Select your Minecraft installation
  4. 4. Click "Install" and wait for completion
  5. 5. Launch Minecraft with Forge profile
  6. 6. Join Pikzels Network for optimal experience
